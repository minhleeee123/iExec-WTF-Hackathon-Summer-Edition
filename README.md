# iExec WTF Hackathon Summer Edition

Workspace dành cho một dự án hackathon định hướng web/mobile app.

## Bắt đầu

1. Đọc `AGENTS.md`.
2. Đọc `PLAN.md` để biết phase hiện tại, approval và bước tiếp theo.
3. Kiểm tra Participation Fit Gate trong `docs/competition-summary.md`.
4. Đọc năm file context trong `docs/` và kiểm tra source ID khi cần.
5. Thực hiện đúng file trong `plan/` tương ứng với phase hiện tại.
6. Chờ người dùng chọn ý tưởng và phê duyệt Product Plan.
7. Chỉ bắt đầu code sau khi Build Plan được phê duyệt.
8. Chỉ xây landing page sau khi core product vượt qua Core Product Ready Gate.

## Thư mục

```text
docs/         Nguồn và context cuộc thi đã chuẩn hóa
PLAN.md       Master execution plan và trạng thái hiện tại
plan/         Research, brainstorm, product và build plan
source-code/  Web/mobile client, landing page và backend/API nếu cần
submission/   Slide và video
```

Submission checklist canonical nằm trong `docs/requirements.md`.
